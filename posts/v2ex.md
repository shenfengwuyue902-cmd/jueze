# [分享创造] 做了个免费决策矩阵工具，专治选择困难症

## 一句话介绍

面对多个选项时不用再凭感觉纠结了 —— 这个工具帮你把决策变成一个量化矩阵，打分排序一目了然。

## 链接

> GitHub: https://github.com/shenfengwuyue902-cmd/jueze
> 在线使用: https://shenfengwuyue902-cmd.github.io/jueze

## 它能做什么

举个栗子：你要买手机，在 3-4 个型号之间纠结。

1. 设定标准：性价比、拍照、续航、品牌 —— 每个分配权重
2. 填上候选机型
3. 给每项打分
4. 工具自动算出加权排名 + 生成柱状图

## 特点

- **单 HTML 文件**，打开即用，无任何依赖
- 数据存在浏览器，关掉再打开还在
- 有暗色模式
- 评分输入有颜色反馈（高分绿、低分橙）
- 排名带进度条，差距一眼看出
- 能导出/导入数据存档
- 结果可以一键复制成一整段文字分享

## 技术层面

纯前端，35KB，Canvas 画柱状图（没用任何第三方库），localStorage 持久化。

部署到 GitHub Pages / Vercel / Netlify 都行。

## 做这个的动机

之前帮朋友挑工作，列了一大堆因素来回比，Excel 又太重。想找个轻量开箱即用的工具发现没有特别满意的，就自己搓了一个。
