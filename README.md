# 决择 · 决策矩阵工具

让每个重要决定都有据可依。

一个轻量级的在线决策矩阵工具，帮助你系统化地比较多个选项，做出理性决策。

---

## 这是什么

面对多个选项时，**决策瘫痪** 是常见困境 —— 选了怕后悔，不选又不甘心。

**决择** 将决策过程转化为一个可量化的矩阵：
1. 设定评估标准并分配权重
2. 列出所有待比较的选项
3. 对每个选项按标准逐一评分
4. 自动计算加权得分，生成排名与图表

## 在线使用

直接从浏览器打开 \index.html\ 即可使用，无需安装、无需服务器。

## 功能特点

- **零依赖** —— 单 HTML 文件，打开即用
- **数据持久化** —— localStorage 自动保存，关闭后数据不丢失
- **暗色/亮色主题** —— 适配系统偏好或手动切换
- **评分视觉反馈** —— 高分绿、中分黄、低分橙，一目了然
- **进度条排名** —— 可视化展示各选项得分差距
- **排序调整** —— 标准和选项支持上下移动
- **交互式步骤导航** —— 点击步骤名称直达对应区域
- **统计概览** —— 最高分、最低分、分差一目了然
- **导出/导入** —— JSON 格式存档与恢复
- **一键复制结果** —— 格式化文本，方便分享
- **响应式设计** —— 手机、平板、桌面均适配

## 使用场景

| 场景 | 示例 |
|------|------|
| 购物决策 | 比较不同品牌的产品 |
| 求职选择 | 评估不同的工作 Offer |
| 选校/选专业 | 对比不同学校或专业方向 |
| 方案评审 | 团队内评估多个方案 |
| 优先级排序 | 对多个事项排定优先级 |
| 生活决策 | 租房、买车、保险方案等 |

## 技术栈

- 纯 HTML + CSS + JavaScript
- Canvas 2D API 绘制柱状图
- localStorage 持久化存储
- 总计约 35KB
